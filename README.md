## Lifecycle
<p>✅ 마운트(Mount): 컴포넌트가 생성되어 DOM에 추가되는 단계</p>
<p>✅ 업데이트(Update): 상태(state)나 props가 변경되어 다시 렌더링되는 단계</p>
<p>✅ 언마운트(Unmount): 컴포넌트가 제거되는 단계</p>
